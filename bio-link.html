How it works: .container: This is the wrapper around the two buttons, making them easy to position relative to each other. .main-button: This is the main button that the user hovers over. .slide-button: This is the hidden button that will slide out when hovering over the main button. Initially, it is positioned off-screen to the right (right: -120px). :hover effect: When the .container (which includes both buttons) is hovered, the .slide-button slides in from the right with transform: translateX(-120px).
